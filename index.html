Módulo 2 — input() = Datos de entrada

input() detiene el programa y espera que el usuario escriba algo. El texto ingresado se guarda en una variable.

Módulo 3 — print() = Datos de salida

print() muestra información en pantalla. Es la forma en que Python entrega resultados al usuario.
>>> Esperando print()...
print("Hola, NeuroBiz") # Muestra: Hola, NeuroBiz
💬

Módulo 4 — Mostrar un mensaje en pantalla

La forma más simple de print(): mostrar un texto fijo (un string entre comillas).

Módulo 5 — Mostrar mensaje + valor de variable

Usamos f-strings (f"...") para combinar texto fijo con el valor de una variable. La {variable} se reemplaza por su valor.
tasa 78.5
→
>>> Presiona Ejecutar ↓
tasa = 78.5 print(f"La tasa de retención es: {tasa}%") # Muestra: La tasa de retención es: 78.5%

Módulo 6 — Operaciones básicas

Python puede hacer cálculos con variables numéricas. Los operadores básicos:
+ suma
847 + 112 = 959
- resta
847 - 112 = 735
* multiplicación
78.5 * 2 = 157.0
/ división
847 / 10 = 84.7
// div. entera
847 // 10 = 84
% residuo
847 % 10 = 7
** potencia
2 ** 3 = 8
= ?
sesiones_asistidas = 847 sesiones_fallidas = 112 total = sesiones_asistidas + sesiones_fallidas # 959 tasa = (sesiones_asistidas / total) * 100 # 88.3%
