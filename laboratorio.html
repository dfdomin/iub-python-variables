🧠 ¿Qué es una variable?

Una variable es un espacio con nombre donde guardas un valor. Como una caja, un frasco o un casillero.

📦 Metáfora 1: La variable como caja etiquetada

nombre de la variable
pi
3.1416
valor guardado
3.1416
→
pi
?
pi = 3.1416

🧪 Metáfora 2: La variable como recipiente

El frasco "tasa" contiene
78.5%

Así como un frasco guarda líquido, una variable guarda un dato. El valor puede cambiar (vaciar, rellenar).

🗄️ Metáfora 3: La variable como casillero

Casillero "documento"
10456789

Cada casillero tiene una etiqueta (nombre) y guarda una sola cosa (valor). Puedes cambiar lo que hay dentro.

🔤 ¿Comillas o sin comillas?

En Python, "847" es texto (string) y 847 es número (int). No son lo mismo.

🔍 ¿Qué pasó en el Experimento 2?

Ejecutaste esto y Python mostró un error. Vamos a leerlo parte por parte para que nunca más te asuste.

🧠 Anatomía del error — apréndelo de memoria

Traceback (most recent call last):
File "<stdin>", line 3
TypeError: can only concatenate str (not "int") to str
📁 Archivo "<stdin>" = entrada estándar (lo que escribiste)
📍 Línea línea 3 → print(a + b) es donde falló
🏷️ Tipo TypeError = estás usando tipos incompatibles
💬 Mensaje "solo se puede concatenar str (no int) con str"

🗣️ Traducción al español coloquial

Python te está diciendo:

"Oye, "847" es texto (por las comillas) y 10 es número. No sé si quieres sumarlos como matemáticas o pegarlos como palabras. No puedo adivinar. Sé claro."

💼 ¿Esto para qué me sirve en Gestión Administrativa?

Interpretar mensajes de error no es solo para programar. Es una habilidad profesional que usaras todos los días:

📋 Ejemplo 1 — Error en un sistema de nómina

El sistema dice: "El valor ingresado en 'horas extra' no es numérico". Si aprendiste a leer errores, sabes que alguien escribió texto donde debía ir un número. Diagnosticas en segundos en vez de llamar a soporte.

📊 Ejemplo 2 — Reporte de inventario rechazado

El ERP devuelve: "No se puede calcular total: campo 'cantidad' contiene caracteres no válidos". Vas directo a la celda del Excel que tiene mal el dato. No necesitas a sistemas para arreglarlo.

🔐 Ejemplo 3 — Falla en facturación electrónica

La DIAN responde: "Error 422: NIT del receptor no cumple formato". Leer el código y el mensaje te dice exactamente qué corregir: el NIT. No pierdes tiempo adivinando.

🔑 Leer errores = diagnosticar rápido = resolver sin depender de otros.

🧪 Ahora tú: diagnostica estos 3 errores

Lee cada mensaje de error y selecciona la causa correcta.

NameError: name 'tasa' is not defined
a) La variable no se puede llamar "tasa"
b) Usaste la variable antes de crearla con =
c) Hay que escribir tasa = 0 al final del programa
TypeError: unsupported operand type(s) for +: 'int' and 'str'
a) No se pueden sumar int y str — uno es número, otro texto
b) El símbolo + no existe en Python
c) Hay que importar la librería math
SyntaxError: EOL while scanning string literal
a) El programa es muy largo
b) El archivo tiene un virus
c) Dejaste comillas sin cerrar — te faltó una " al final
