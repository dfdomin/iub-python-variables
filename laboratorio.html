🧠 ¿Qué es una variable?

Una variable es un espacio con nombre donde guardas un valor. Como una caja, un frasco o un casillero.

📦 Metáfora 1: La variable como caja etiquetada

nombre de la variable
pi
3.1416
valor guardado
3.1416
→
pi
?
pi = 3.1416

🧪 Metáfora 2: La variable como recipiente

El frasco "tasa" contiene
78.5%

Así como un frasco guarda líquido, una variable guarda un dato. El valor puede cambiar (vaciar, rellenar).

🗄️ Metáfora 3: La variable como casillero

Casillero "documento"
10456789

Cada casillero tiene una etiqueta (nombre) y guarda una sola cosa (valor). Puedes cambiar lo que hay dentro.

🔤 ¿Comillas o sin comillas?

En Python, "847" es texto (string) y 847 es número (int). No son lo mismo.
