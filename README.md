# 🐍 Python para Negocios — Repaso Parcial

App interactiva para preparar el parcial de Fundamentos de Computación (IUB).

## Módulos

1. **¿Qué es una variable?** — Caja etiquetada visual
2. **input()** — Simulación de entrada de datos
3. **print()** — Salida en pantalla virtual
4. **Mensaje en pantalla** — print() con strings fijos
5. **Mensaje + variable** — f-strings con interpolación visual
6. **Operaciones básicas** — Calculadora interactiva (+, -, *, /, //, %, **)
7. **Autoevaluación** — 5 preguntas de selección múltiple

## Stack

- HTML/CSS/JS vanilla (sin framework)
- Supabase (progreso de estudiantes)
- GitHub Pages (deploy)
- Mobile-first (estudiantes en celular)
